documentation.

## Maintenance

- Keep the English and Chinese trees aligned by topic and intent.
- Update these docs when exported SDK behavior, examples, model adapters, or
  production recommendations change.
- Keep runnable code in `examples/` and reference those examples from the docs.
- Keep internal planning material under `docs/superpowers/`; user-facing SDK
  documentation belongs here.
